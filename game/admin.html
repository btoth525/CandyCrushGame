What this does

  • Push sends your current local config (everything from the other tabs) to the server. The server saves it as /data/config.json. From then on, every visitor's game fetches that file on boot and uses it as the base. Their per-browser tweaks still layer on top.
  • Pull replaces your local config with whatever's currently live on the server. Useful if you want to edit what's published.
  • Clear removes the server config; visitors fall back to bundled defaults.
Changes apply after Save. Use ⌘/Ctrl+S as a shortcut.
